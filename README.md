# GithubAction

* 빌드 및 배포를 자동화 하기위한 스크립트 작성
* AWS EC2, S3, CodeDeploy, IAM, RDS 사용
* CI/CD 개념 학습
* AWS 와 Github Action 을 사용하여 CI/CD 프로젝트 구축

## CI(Coninuous Integration)
* 자동으로 프로젝트에 코드가 머지된다.

## CD(Continuous Deployment)
* 자동으로 코드가 배포된다.

## CI/CD가 필요한 이유
* 개발을 좀 더 수월하게 하기 위함
* 개발에 좀 더 집중 가능
* 시간을 아낄 수 있다(=반복되는 행위를 없기때문)
